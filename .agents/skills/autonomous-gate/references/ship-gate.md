# Ship Gate Prompts

Use only concerns supported by the release scope, verified evidence, and the
project's deployment constraints.

- Have the required earlier gates passed and been linked?
- Is the relevant verification evidence complete and current?
- Are migration, configuration, monitoring, rollback, and ownership needs
  addressed when applicable?
- Is the release boundary and target environment explicit?
- Does the evidence support a release decision without waived unknowns?
